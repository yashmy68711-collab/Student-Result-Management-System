# Python Student Result Management System

## About

A CLI Student Result Management System built using Python.
This application stores student marks, calculates total marks, percentage, and grade automatically.

---

## ✨ Features

✔️ Add student records
✔️ Calculate total marks
✔️ Calculate percentage
✔️ Assign grades automatically
✔️ View all student results
✔️ Search student records
✔️ Input validation

---

## 🛠️ Tech Used

* Python
* Object-Oriented Programming (OOP)
* Dictionaries


## 🚀 How to Run

1. Download the project
2. Open the Python file
3. Run the program
4. Choose options from the menu
5. Add or view student results
